A very simple WordPress plugin shows the recent tweets from user's Sina Weibo.

== Description ==

It is a very simple WordPress plugin shows the recent tweets from user's Sina Weibo.

The features:

*   It has a widget, you can drag it drop it where you want in the side.Also you can set it how many tweetes you want to show.
*   You do not need to input you Sina account and password, you just tell Sina that you want to use this plugin by click a link.

== Installation ==

1. Upload the directory `sinaweibo` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
